== Installation ==

1.	WP Sugar  is very easy to install like as other simple Wordpress plugins. No need to edit or modify anything here in code.
2.	Unzip the file 'wp-sugar-free.zip'.                              
3.	Upload the 'wp-sugar-free' directory to '/wp-content/plugins/' directory using ftp client
4.	Or you can upload and install wp-sugar-free.zip through plugin install wizard in WP admin panel.
5.	Activate the plugin through the 'Plugins' menu in Wordpress.
6.	After activating, you will see an option for WP Sugar Free in the admin menu (left navigation).

== Frequently Asked Questions ==

How to configure wp-sugar-free?

	Provide your SugarCrm instance URL in url.
	Provide your SugarCrm instance admin users username, password in User Name, Password fields are the username, password of your SugarCRM instance from which you login to your SugarCRM.


How to configure wp-suger-free webform ?

	Go to "Lead Form Fields" link.
	The third column "Show Field" displays the status of the field.
	The first column (the checkbox column) is used to select the field you want to display in the front end).
	After the selecting which are all the fields you want to display in the front end, click "Save Field Settings" button to save the field settings.
	Copy the shortcode display above the page and paste it in page/post
	Go to the frontend page/post in which you pasted the shotcode now you are ready to capture leads.
	If you want to create custom fields, You can create that field in your SugarCRM instance it will be fetched to wp-tiger-free then you can activate the field.
